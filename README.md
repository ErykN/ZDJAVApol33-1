# ZDJAVApol33

# Przygotowanie katalogów

Tworzenie folderu na projekty:
```cmd
cd Desktop
```
lub
```cmd
cd Pulpit
```
następnie
```cmd
mkdir projects
```

Tworzenie folderu dla git:
```cmd
mkdir git
```

# Repozytoria
Wejście do folderu git:
```cmd
cd git
```

Projekt z zajęć:
```cmd
git clone https://github.com/jakub-olszewski/ZDJAVApol33.git
```
Mój projekt:
```cmd
git clone https://github.com/jakub-olszewski/my-project.git
```

# Dodanie do Intellij

Stworzenie "Empty project"
```cmd
File -> New -> Project -> Empty Project
```
nazwa "ZDJAVApol33"

Dodanie modułów
```cmd
File -> New -> Module from Existing Sources...
```
Wybieramy folder
odpowiednio 
```cmd
git/ZDJAVApol33
git/my-project
```
